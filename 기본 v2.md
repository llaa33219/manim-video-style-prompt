manim과 일레븐랩스 tts를 이용하여 영상을 제작한다.
일레븐랩스 api key는 사용자가 알려줄 것이다.
기본적으로 흰 배경에 이미지/영상이 하나 존재하며, 아래에는 투명도 25%의 검정 배경에 흰 글자로 자막이 달려있는 스타일로 한다.
이미지/영상 소스는 사용자가 제공한 소스로 사용하거나, 직접 이미지를 찾아 다운로드하여 사용한다. manim과 svg등으로 직접 이미지를 직접 제작하는 것도 허용되긴 하지만, 데이터 표기를 위한 경우를 제외하면, 극도로 권장되지 않는다.
이미지/영상은 실사용 녹화, 밈, 로고, 실사용 이미지, 화면 캡쳐, 결과물 캡쳐 등 여러 형식의 이미지가 모두 허용된다. 밈 gif와 같이 재미를 위한 이미지/영상은 허용하며 권장되지만, 쓸대없이 복잡한 이미지/영상 사용은 엄격히 금지한다.
자막은 심각한 문제가 있지 않는 한, 영상 중앙의 하단에 배치하며, 위치 이동을 금지한다. 또한 자막의 스타일은 위에서 명시했듯, 투명도 25%의 검정 배경을 가지고 흰색 글자를 사용하며 https://cdn.jsdelivr.net/gh/projectnoonnu/2408@1.0/goorm-sans-regular.woff2 폰트를 사용한다.
자막의 스타일 변경은 기본적으로 금지하지만, 강조 등의 목적을 위하여 일부 또는 전체의 글자 굵기를 변경하는 것은 허용된다.
영상의 마지막에는 (당신의 모델명)이 manim과 일레븐랩스 tts를 이용하여 영상을 제작하였다고만 알려줘.
